# modulato

Animation-first React framework for marketing sites, landing pages and
portfolios — designed to be the fastest framework for LLMs to build visually
impactful websites.

- Pages coexist during navigation: crossfade, slide, or FLIP shared elements
  between real layouts.
- A persistent shell (menu, cursor, canvas, WebGL) that reacts to the URL.
- Strict mount/destroy lifecycles for everything animated.
- Server-rendered HTML, hydrated once. Motion tokens tweakable live.

```sh
npm create modulato@latest my-site
```

The complete reference lives in every scaffolded project as `MODULATO.md` —
one file, written for humans and language models alike.
