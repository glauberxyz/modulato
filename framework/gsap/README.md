# @modulato/gsap

GSAP glue for Modulato: `useMotion` runs your animation code inside a
`gsap.context()` scoped to the page element — auto-reverted on unmount,
re-run on breakpoint changes and Tweak Mode replays.

Part of [modulato](https://www.npmjs.com/package/modulato) — scaffold a site
with `npm create modulato@latest`.
