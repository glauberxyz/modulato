# @modulato/vite

The Modulato Vite plugin: file-based routing from `pages/`, dev SSR
middleware, production client+SSR builds, Vercel Build Output emission,
style auto-import, motion-token registration and Tweak Mode wiring.

Part of [modulato](https://www.npmjs.com/package/modulato) — scaffold a site
with `npm create modulato@latest`.
