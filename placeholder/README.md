# Modulato

**Animation-first React framework** for marketing sites, landing pages and portfolios —
page transitions, shared-element FLIP, per-page intros, and choreography as a first-class
citizen.

> ⚠️ This version is a **name-claim placeholder**. The framework is in active
> development and not yet usable. Watch [modulato.org](https://modulato.org) for the
> first real release.

## What's coming

- File-based pages with zero registration — one folder per page
- Transitions between pages as small, colocated pair files
- Shared-element FLIP transitions built in
- Per-page first-load intro animations
- SSR for SEO, hydrated once, animated forever
- GSAP as the animation engine
- Designed to be the fastest framework for LLMs to build visually impactful sites
