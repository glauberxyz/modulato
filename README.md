# Modulato

An animation-first React framework for marketing sites, landing pages, and portfolios —
designed to be the fastest framework for LLMs to build visually impactful websites.

See [PLAN.md](./PLAN.md) for the full design document.

## Structure

- `framework/` — the publishable packages (`modulato`, `@modulato/vite`, `@modulato/server`, …)
- `examples/demo` — the 3-page proving-ground site
- `docs/` — LLM-first documentation (source of truth: `MODULATO.md`)

## Development

```sh
npm i
npm run dev     # starts examples/demo with SSR + HMR
```
